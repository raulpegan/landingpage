Intro

Raul Pegan's personal website

This is my personal website where I will be posting all information pertaining to my career and my hobbies, oftentimes they are two in the same. This simple site is still under construction, and I hope to link all my personal projects and career experience to it.

My approach to the design of this page is a minimalist one. I want to reduce the visual clutter as much as possible, as well as writing very simple code. Therefore, this site might not be compatible with deprecated browsers. Also it looks nice with my Arch rice.

The website itself is a personal project, so do not be surprised if you see some very awful styling and bad coding standards, this is where I put my css and html to a test.

Additionally, I am hosting this on a static hosting provider, so I'm using plain css, js, and html. Following the minimalist philosophy, no fancy, clutter-filled, heavy web frameworks. Expect routing horrors.
